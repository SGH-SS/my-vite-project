üìã Setup Steps

Step 1: Request Access

Visit TradingView Advanced Charts and fill out the access request form.

Note: Approval typically takes 1-2 business days.

Step 2: Download Library

After approval, download the library ZIP file from the provided GitHub repository.

Step 3: Extract Files

Extract the contents to your project's public directory:

mkdir -p public/charting_library # Extract ZIP contents to public/charting_library/

Step 4: Add Script Tags

Add the required script tags to your index.html file:

<!-- Add to your index.html head section --> <script src="/charting_library/charting_library.standalone.js"></script> <!-- Optional: For demo data --> <script src="/charting_library/datafeeds/udf/dist/bundle.js"></script>

Step 5: Use AdvancedChart Component

Import and use the AdvancedChart component in your React app:

import AdvancedChart from './components/AdvancedChart'; // In your component <AdvancedChart isDarkMode={isDarkMode} chartData={chartData} onCandleSelect={handleCandleSelect} />
